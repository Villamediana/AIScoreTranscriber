What is AI Score Transcriber?

Sometimes we can't find the piano score for a specific piece, which makes practice harder. AI Score Transcriber helps with that.

Upload an audio file and the app generates a piano roll view and MIDI file to practice with. It works like a piano tutorial for any piece.

Works best with instrumental or piano music, but you can try any audio.
